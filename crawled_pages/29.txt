<h1 id="firstHeading" class="firstHeading" lang="en">DBLP</h1>			<div id="bodyContent" class="mw-body-content">

<p><b>DBLP</b> is a <a href="/wiki/Computer_science" title="Computer science">computer science</a> <a href="/wiki/Bibliography" title="Bibliography">bibliography</a> website. Starting in 1993 at the <a href="/wiki/University_of_Trier" title="University of Trier">University of Trier</a>, <a href="/wiki/Germany" title="Germany">Germany</a>, it grew from a small collection of HTML files<sup id="cite_ref-Ley2009_1-0" class="reference"><a href="#cite_note-Ley2009-1">&#91;1&#93;</a></sup> and became an organization hosting a <a href="/wiki/Database" title="Database">database</a> and <a href="/wiki/Logic_programming" title="Logic programming">logic programming</a> bibliography site. DBLP listed more than 3.66 million journal articles, conference papers, and other publications on computer science in July 2016, up from about 14,000 in 1995.<sup id="cite_ref-2" class="reference"><a href="#cite_note-2">&#91;2&#93;</a></sup> All important journals on computer science are tracked. Proceedings papers of many conferences are also tracked. It is mirrored at three sites across the <a href="/wiki/Internet" title="Internet">Internet</a>.<sup id="cite_ref-dblp_3-0" class="reference"><a href="#cite_note-dblp-3">&#91;3&#93;</a></sup><sup id="cite_ref-4" class="reference"><a href="#cite_note-4">&#91;4&#93;</a></sup><sup id="cite_ref-5" class="reference"><a href="#cite_note-5">&#91;5&#93;</a></sup>
</p><p>For his work on maintaining DBLP, <a href="/wiki/Michael_Ley" title="Michael Ley">Michael Ley</a> received an award from the <a href="/wiki/Association_for_Computing_Machinery" title="Association for Computing Machinery">Association for Computing Machinery</a> and the VLDB Endowment Special Recognition Award in 1997.
</p><p><i>DBLP</i> originally stood for <i>DataBase systems and Logic Programming</i>. As a <a href="/wiki/Backronym" title="Backronym">backronym</a>, it has been taken to stand for <i>Digital Bibliography &amp; Library Project</i>;<sup id="cite_ref-6" class="reference"><a href="#cite_note-6">&#91;6&#93;</a></sup> however, it is now preferred that the acronym be simply a name, hence the new title "The DBLP Computer Science Bibliography".<sup id="cite_ref-dblp-faq_7-0" class="reference"><a href="#cite_note-dblp-faq-7">&#91;7&#93;</a></sup>
</p><p>Users of dblp remain unaffected by some additional attributes in the DTD as of February 2016, which are meant to support future versions of the data file. Thus consumers of the raw dblp.xml file should update their local dblp.dtd file, according to the notice on the home page, as of February 2016.<sup id="cite_ref-8" class="reference"><a href="#cite_note-8">&#91;8&#93;</a></sup>
</p>

<p><i>DBL-Browser</i> (<i>Digital Bibliographic Library Browser</i>) is a utility for browsing the DBLP website. The browser was written by Alexander Weber in 2005 at the <a href="/wiki/University_of_Trier" title="University of Trier">University of Trier</a>. It was designed for use off-line in reading the DBLP, which consisted of 696,000 bibliographic entries in 2005 (and in 2015 has more than 2.9 million).
</p><p>DBL-Browser is <a href="/wiki/GPL" class="mw-redirect" title="GPL">GPL</a> <a href="/wiki/Software" title="Software">software</a>, available for download from <a href="/wiki/SourceForge" title="SourceForge">SourceForge</a>. It uses the <a href="/wiki/XML" title="XML">XML</a> <a href="/wiki/Document_Type_Definition" class="mw-redirect" title="Document Type Definition">DTD</a>. Written in <a href="/wiki/Java_programming_language" class="mw-redirect" title="Java programming language">Java programming language</a>, this code shows the bibliographic entry in several types of screens, ranging from graphics to text:
</p>
<ul><li>Author page</li>
<li>Article page</li>
<li>Table of contents</li>
<li>Related conferences / journals</li>
<li>Related authors (graphic representation of relationships)</li>
<li>Trend analysis (graphics histogram)</li></ul>
<p>DBLP is similar to the bibliographic portion of <a href="/wiki/Arxiv.org" class="mw-redirect" title="Arxiv.org">arxiv.org</a> which also links to articles. DBL-Browser provides a means to view some of the associated computer science articles.
</p>

<ul><li><span class="official-website"><span class="url"><a rel="nofollow" class="external text" href="https://dblp.org/">Official website</a></span></span> <a href="https://www.wikidata.org/wiki/Q1224715#P856" title="Edit this at Wikidata"><img alt="Edit this at Wikidata" src="//upload.wikimedia.org/wikipedia/commons/thumb/7/73/Blue_pencil.svg/10px-Blue_pencil.svg.png" width="10" height="10" style="vertical-align: text-top" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/7/73/Blue_pencil.svg/15px-Blue_pencil.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/7/73/Blue_pencil.svg/20px-Blue_pencil.svg.png 2x" data-file-width="600" data-file-height="600" /></a>
<ul><li><a rel="nofollow" class="external text" href="https://dblp.org/search/">CompleteSearch DBLP</a> provides a fast search-as-you-type interface to DBLP, as well as faceted search. It is maintained by <a href="/wiki/Hannah_Bast" title="Hannah Bast">Hannah Bast</a> and synchronized twice daily with the DBLP database. Since December 2007, the search functionality is embedded into each DBLP author page (via <a href="/wiki/Javascript" class="mw-redirect" title="Javascript">Javascript</a>).</li></ul></li>
<li><cite class="citation web"><a rel="nofollow" class="external text" href="https://www.dagstuhl.de/en/about-dagstuhl/projects/lzi-dblp/">"LZI+DBLP"</a>. <i><a href="/wiki/Schloss_Dagstuhl" class="mw-redirect" title="Schloss Dagstuhl">Schloss Dagstuhl</a></i>. <a href="/wiki/Leibniz_Center_for_Informatics" class="mw-redirect" title="Leibniz Center for Informatics">Leibniz Center for Informatics</a><span class="reference-accessdate">. Retrieved <span class="nowrap">2017-02-11</span></span>.</cite><span title="ctx_ver=Z39.88-2004&amp;rft_val_fmt=info%3Aofi%2Ffmt%3Akev%3Amtx%3Ajournal&amp;rft.genre=unknown&amp;rft.jtitle=Schloss+Dagstuhl&amp;rft.atitle=LZI%2BDBLP&amp;rft_id=https%3A%2F%2Fwww.dagstuhl.de%2Fen%2Fabout-dagstuhl%2Fprojects%2Flzi-dblp%2F&amp;rfr_id=info%3Asid%2Fen.wikipedia.org%3ADBLP" class="Z3988"></span><link rel="mw-deduplicated-inline-style" href="mw-data:TemplateStyles:r861714446"/></li>
<li><a rel="nofollow" class="external text" href="https://web.archive.org/web/20110222024555/http://dbis.uni-trier.de/DBL-Browser/">DBL-Browser</a> at the <a href="/wiki/Wayback_Machine" title="Wayback Machine">Wayback Machine</a>&#32;(archived 2011-02-22)</li>
<li><a rel="nofollow" class="external text" href="http://dblp.L3S.de">FacetedDBLP</a> provides a faceted search interface to DBLP, synchronized once per week with the DBLP database. In addition to common facets such as year, author, or venues, it contains a topic-based facet summarizing and characterizing the current result set based on the author keywords for individual publications. For the DBLP data, FacetedDBLP also provides an <a href="/wiki/Resource_Description_Framework" title="Resource Description Framework">RDF</a> dump (using <a rel="nofollow" class="external text" href="http://www4.wiwiss.fu-berlin.de/bizer/d2r-server/">D2R server</a> technology) as well as an <a href="/wiki/SQL" title="SQL">SQL</a> dump based on the underlying mysql database.</li>
<li><a rel="nofollow" class="external text" href="http://www.confsearch.org">confsearch</a> Conference search engine and calendar based on DBLP.</li>
<li><a rel="nofollow" class="external text" href="http://dblp.cloudmining.net/">CloudMining DBLP</a> is another faceted search solution with different visualizations.</li></ul>

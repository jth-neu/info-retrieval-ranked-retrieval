<h1 id="firstHeading" class="firstHeading" lang="en">W3Catalog</h1>			<div id="bodyContent" class="mw-body-content">
				<div id="siteSub" class="noprint">From Wikipedia, the free encyclopedia</div>				<div id="contentSub"></div>
				<div id="jump-to-nav"></div>				<a class="mw-jump-link" href="#mw-head">Jump to navigation</a>
				<a class="mw-jump-link" href="#p-search">Jump to search</a>
				<div id="mw-content-text" lang="en" dir="ltr" class="mw-content-ltr"><div class="mw-parser-output"><p><b>W3 Catalog</b> was a very early <a href="/wiki/Web_search_engine" title="Web search engine">web search engine</a>, first released on September 2, 1993<sup id="cite_ref-Announcement_1-0" class="reference"><a href="#cite_note-Announcement-1">&#91;1&#93;</a></sup> by developer <a href="/wiki/Oscar_Nierstrasz" title="Oscar Nierstrasz">Oscar Nierstrasz</a> at the <a href="/wiki/University_of_Geneva" title="University of Geneva">University of Geneva</a>.
</p><p>Unlike later search engines, like <a href="/wiki/Aliweb" title="Aliweb">Aliweb</a>, which attempt to index the web by crawling over the accessible content of web sites, W3 Catalog exploited the fact that many high-quality, manually maintained lists of web resources were already available. W3 Catalog simply mirrored these pages, reformatted the contents into individual entries, and provided a <a href="/wiki/Perl" title="Perl">Perl</a>-based front-end to enable dynamic querying.<sup id="cite_ref-history_2-0" class="reference"><a href="#cite_note-history-2">&#91;2&#93;</a></sup><sup id="cite_ref-virtual_3-0" class="reference"><a href="#cite_note-virtual-3">&#91;3&#93;</a></sup>
</p><p>At the time, <a href="/wiki/Common_Gateway_Interface" title="Common Gateway Interface">CGI</a> did not yet exist, so W3 Catalog was implemented as an extension to Tony Sander's Plexus web server, implemented in <a href="/wiki/Perl" title="Perl">Perl</a>.
</p><p>W3 Catalog was retired on December 8 1996.<sup id="cite_ref-history_2-1" class="reference"><a href="#cite_note-history-2">&#91;2&#93;</a></sup>
</p>

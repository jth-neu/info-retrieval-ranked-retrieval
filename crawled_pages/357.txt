<p><b>OpenSearch</b> is a collection of technologies that allow publishing of search results in a format suitable for <a href="/wiki/Web_syndication" title="Web syndication">syndication</a> and <a href="/wiki/Search_aggregator" title="Search aggregator">aggregation</a>. It is a way for <a href="/wiki/Website" title="Website">websites</a> and <a href="/wiki/Search_engine" class="mw-redirect" title="Search engine">search engines</a> to publish search results in a standard and accessible format.
</p><p>OpenSearch was developed by <a href="/wiki/Amazon.com" class="mw-redirect" title="Amazon.com">Amazon.com</a> subsidiary <a href="/wiki/A9.com" title="A9.com">A9</a> and the first version, OpenSearch 1.0, was unveiled by <a href="/wiki/Jeff_Bezos" title="Jeff Bezos">Jeff Bezos</a> at the <a rel="nofollow" class="external text" href="https://conferences.oreilly.com/etech">O'Reilly Emerging Technology Conference</a> in March, 2005. Draft versions of OpenSearch 1.1 were released during September and December 2005. The OpenSearch specification is licensed by A9 under the <a href="/wiki/Creative_Commons" title="Creative Commons">Creative Commons</a> Attribution-ShareAlike 2.5 License.<sup id="cite_ref-1" class="reference"><a href="#cite_note-1">&#91;1&#93;</a></sup>
</p>

<h2><span class="mw-headline" id="Support">Support</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=OpenSearch&amp;action=edit&amp;section=1" title="Edit section: Support">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/Web_browser" title="Web browser">Web browsers</a> that support OpenSearch include <a href="/wiki/Safari_(web_browser)" title="Safari (web browser)">Safari</a>,<sup id="cite_ref-2" class="reference"><a href="#cite_note-2">&#91;2&#93;</a></sup> <a href="/wiki/Microsoft_Edge" title="Microsoft Edge">Microsoft Edge</a>,<sup id="cite_ref-3" class="reference"><a href="#cite_note-3">&#91;3&#93;</a></sup> <a href="/wiki/Firefox" title="Firefox">Firefox</a><sup id="cite_ref-4" class="reference"><a href="#cite_note-4">&#91;4&#93;</a></sup> and <a href="/wiki/Google_Chrome" title="Google Chrome">Google Chrome</a>.<sup id="cite_ref-5" class="reference"><a href="#cite_note-5">&#91;5&#93;</a></sup>
</p>
<h2><span class="mw-headline" id="Design">Design</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=OpenSearch&amp;action=edit&amp;section=2" title="Edit section: Design">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="thumb tright"><div class="thumbinner" style="width:302px;"><a href="/wiki/File:Dewiki-suggesting-searchplugin.png" class="image"><img alt="" src="//upload.wikimedia.org/wikipedia/commons/thumb/c/cc/Dewiki-suggesting-searchplugin.png/300px-Dewiki-suggesting-searchplugin.png" width="300" height="158" class="thumbimage" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/c/cc/Dewiki-suggesting-searchplugin.png/450px-Dewiki-suggesting-searchplugin.png 1.5x, //upload.wikimedia.org/wikipedia/commons/c/cc/Dewiki-suggesting-searchplugin.png 2x" data-file-width="584" data-file-height="308" /></a>  <div class="thumbcaption"><div class="magnify"><a href="/wiki/File:Dewiki-suggesting-searchplugin.png" class="internal" title="Enlarge"></a></div>Search suggestions in the <a href="/wiki/German_Wikipedia" title="German Wikipedia">German Wikipedia</a></div></div></div>
<p>OpenSearch consists of:
</p>
<ol><li>OpenSearch Description files: <a href="/wiki/Extensible_Markup_Language" class="mw-redirect" title="Extensible Markup Language">XML</a> files that identify and describe a search engine.</li>
<li>OpenSearch Query Syntax: describe where to retrieve the search results</li>
<li>OpenSearch <a href="/wiki/RSS_(file_format)" class="mw-redirect" title="RSS (file format)">RSS</a> (in OpenSearch 1.0) or OpenSearch Response (in OpenSearch 1.1): format for providing open search results.</li>
<li>OpenSearch <a href="/wiki/Search_aggregator" title="Search aggregator">Aggregators</a>: Sites that can display OpenSearch results.</li>
<li>OpenSearch "Auto-discovery" to signal the presence of a search plugin link to the user and the link embedded in the header of HTML pages</li></ol>
<p>OpenSearch Description Documents list search result responses for the given website/tool. Version 1.0 of the specification only allowed one response, in RSS format; however, version 1.1 provides support for multiple responses, which may be in any format. RSS and <a href="/wiki/Atom_(standard)" class="mw-redirect" title="Atom (standard)">Atom</a> are the only ones formally supported by OpenSearch aggregators, however other types, such as <a href="/wiki/HyperText_Markup_Language" class="mw-redirect" title="HyperText Markup Language">HTML</a> are perfectly acceptable.
</p>
<ul><li>Auto-discovery of an OpenSearch Description Document is available from both HTML and Atom or RSS feed documents via <a href="/wiki/Link_relations" class="mw-redirect" title="Link relations">Link relations</a> in the form of <code>&lt;atom:link rel="search" ... /&gt;</code> for Atom feeds<sup id="cite_ref-:0_6-0" class="reference"><a href="#cite_note-:0-6">&#91;6&#93;</a></sup> or <code>&lt;link rel="search" ... /&gt;</code> for RSS feeds<sup id="cite_ref-:0_6-1" class="reference"><a href="#cite_note-:0-6">&#91;6&#93;</a></sup> and HTML documents.<sup id="cite_ref-7" class="reference"><a href="#cite_note-7">&#91;7&#93;</a></sup></li>
<li>OpenSearch Description Document must be placed on a web server of the same domain.<sup id="cite_ref-8" class="reference"><a href="#cite_note-8">&#91;8&#93;</a></sup></li>
<li>OpenSearch Description Documents must be served with the <code>application/opensearchdescription+xml</code> <a href="/wiki/Internet_media_type" class="mw-redirect" title="Internet media type">Internet media type</a>.<sup id="cite_ref-9" class="reference"><a href="#cite_note-9">&#91;9&#93;</a></sup></li></ul>

<h1 id="firstHeading" class="firstHeading" lang="en">Web indexing</h1>			<div id="bodyContent" class="mw-body-content">
				<div id="siteSub" class="noprint">From Wikipedia, the free encyclopedia</div>				<div id="contentSub"></div>

<p><b>Web indexing</b> (or <b>Internet indexing</b>) refers to various methods for indexing the contents of a <a href="/wiki/Website" title="Website">website</a> or of the <a href="/wiki/Internet" title="Internet">Internet</a> as a whole. Individual websites or <a href="/wiki/Intranet" title="Intranet">intranets</a> may use a <a href="/wiki/Back-of-the-book_index" class="mw-redirect" title="Back-of-the-book index">back-of-the-book index</a>, while <a href="/wiki/Search_engines" class="mw-redirect" title="Search engines">search engines</a> usually use keywords and <a href="/wiki/Metadata_(computing)" class="mw-redirect" title="Metadata (computing)">metadata</a> to provide a more useful vocabulary for Internet or onsite searching. With the increase in the number of <a href="/wiki/Periodical" class="mw-redirect" title="Periodical">periodicals</a> that have articles online, web indexing is also becoming important for periodical websites.<sup id="cite_ref-1" class="reference"><a href="#cite_note-1">&#91;1&#93;</a></sup>
</p><p>Back-of-the-book-style web indexes may be called "web site A-Z indexes".<sup id="cite_ref-2" class="reference"><a href="#cite_note-2">&#91;2&#93;</a></sup> The implication with "A-Z" is that there is an alphabetical browse view or interface. This interface differs from that of a browse through layers of hierarchical categories (also known as a <a href="/wiki/Taxonomy_(general)" title="Taxonomy (general)">taxonomy</a>) which are not necessarily alphabetical, but are also found on some web sites. Although an A-Z index could be used to index multiple sites, rather than the multiple pages of a single site, this is unusual.
</p><p>Metadata web indexing involves assigning keywords or phrases to web pages or web sites within a <a href="/wiki/Metadata_tag" class="mw-redirect" title="Metadata tag">metadata tag</a> (or "meta-tag") field, so that the web page or web site can be retrieved with a search engine that is customized to search the keywords field.<sup id="cite_ref-3" class="reference"><a href="#cite_note-3">&#91;3&#93;</a></sup> This may or may not involve using keywords restricted to a controlled vocabulary list. This method is commonly used by <a href="/wiki/Search_engine_indexing" title="Search engine indexing">search engine indexing</a>.
</p>

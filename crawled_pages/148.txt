<p><b>Picollator</b> - <a href="/wiki/Internet_search_engine" class="mw-redirect" title="Internet search engine">Internet search engine</a> that performs search for web sites and <a href="/wiki/Multimedia" title="Multimedia">multimedia</a> by visual query (image) or text, or a combination of visual query and text. Picollator recognizes objects in the image, obtains their relevance to the text and vice a versa, and searches in accordance with all information provided.
</p>

<h2><span class="mw-headline" id="Description">Description</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Picollator&amp;action=edit&amp;section=1" title="Edit section: Description">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>At present Picollator identifies human faces on the images and creates a <a href="/wiki/Database" title="Database">database</a> of people's faces.  So you can search for websites containing photos, other images of the submitted person, lookalikes, similar photos in the Internet.
</p><p>Visual query offers quite a natural approach to search: upload your image and search for similar pictures. This makes the system totally language-independent.
</p><p>Picollator goes to the hybrid search. It allows users to search for webpages by text also, or combine visual and text query. Complex search query (image+text) provides the best possible results, transforming all media into knowledge.
</p>
<h2><span class="mw-headline" id="History">History</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Picollator&amp;action=edit&amp;section=2" title="Edit section: History">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><b>2006</b> Recogmission LLC developed a desktop application for photo collections management. The system automatically classifies, manages and retrieves photographs stored locally or in corporate databases.
</p><p><b>2007</b> Recogmission started Picollator multimedia search engine project, now in Beta stage.
</p><p><b>2008</b> Picollator.mobi <a rel="nofollow" class="external autonumber" href="http://picollator.mobi">[1]</a> - the new universal search engine for mobile phones
</p><p><b>2009</b> Recogmission opens the web based <a href="/wiki/Content_filter" class="mw-redirect" title="Content filter">content filter</a> service piFilter.com <a rel="nofollow" class="external autonumber" href="http://pifilter.com">[2]</a>, which inherited some pattern recognition technologies from Picollator search engine.
</p>
<h2><span class="mw-headline" id="Features">Features</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Picollator&amp;action=edit&amp;section=3" title="Edit section: Features">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Most image search engines match user textual query and picture tags. Picollator is based on a different approach. It stores in its database patterns or objects found in the picture. It is able to understand who is in the photo and compare it to other photos and find similar ones.
</p><p>Picollator is able to perform search by images and/or text to find digital images and/or textual resources in the Internet.
</p><p>To search for multimedia information user may submit
</p>
<ul><li>Sample image to find images with relevant people</li>
<li>Image to find web resources</li>
<li>Text to find images</li>
<li>Text to find web resources</li>
<li>Text and Image to find images and/or web resources.</li></ul>
<h2><span class="mw-headline" id="Facts">Facts</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Picollator&amp;action=edit&amp;section=4" title="Edit section: Facts">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Picollator is the noun, while the corresponding verb 'to picollate' has been first invented by Recogmission team. 'Picollate' consists of two word samples: 'PICture' and 'COLLATE'. Recogmission used the word 'picollate' in the meaning of the process of picture searching and managing.
</p><p>Picollator generally works with digital photos and texts and offers the hybrid search possibility, because the user is able to find any resource using the text or rich content. Yet, the system is able to process artworks and even painted portraits.
</p><p>Picollator image index contains black-and-white photos as well as coloured photographs.
</p>
<h2><span class="mw-headline" id="Company">Company</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Picollator&amp;action=edit&amp;section=5" title="Edit section: Company">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Recogmission <a href="https://ru.wikipedia.org/wiki/Recogmission_LLC" class="extiw" title="ru:Recogmission LLC">ru:Recogmission LLC</a> has developed a novel indexing engine for multimedia information search based on the visual query.
</p><p>Recogmission develops solutions for multimedia information (image, text and video) indexing and search on the Web and in corporate environments.
</p>

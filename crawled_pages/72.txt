<h1 id="firstHeading" class="firstHeading" lang="en">Voice search</h1>			<div id="bodyContent" class="mw-body-content">
				<div id="siteSub" class="noprint">From Wikipedia, the free encyclopedia</div>				<div id="contentSub"></div>
				<div id="jump-to-nav"></div>				<a class="mw-jump-link" href="#mw-head">Jump to navigation</a>
				<a class="mw-jump-link" href="#p-search">Jump to search</a>
				<div id="mw-content-text" lang="en" dir="ltr" class="mw-content-ltr"><div class="mw-parser-output"><p><b>Voice search</b>, also called voice-enabled, allows the user to use a voice command to search the Internet, or a portable device.  Currently, voice search is commonly used in (in a narrow sense) "directory assistance", or local search. Examples include <a href="/wiki/Google_411" class="mw-redirect" title="Google 411">Google 411</a>, <a href="/wiki/Tellme" class="mw-redirect" title="Tellme">Tellme</a> directory assistance and <a href="/wiki/Yellowpages.com" title="Yellowpages.com">Yellowpages.com</a>'s 1-800-YellowPages. 
</p><p>In a broader definition, voice search include open-domain keyword query on any information on the Internet, for example in <a href="/wiki/Google_Voice_Search" title="Google Voice Search">Google Voice Search</a>, <a href="/wiki/Cortana_(software)" class="mw-redirect" title="Cortana (software)">Cortana</a>, <a href="/wiki/Siri" title="Siri">Siri</a> and <a href="/wiki/Amazon_Echo" title="Amazon Echo">Amazon Echo</a>. Given that voice-based systems are interactive, such systems are also called open-domain <a href="/wiki/Question_answering" title="Question answering">question answering</a> systems. 
</p><p>Voice search is often interactive, involving several rounds of interaction that allows a system to ask for clarification. Voice search is a type of <a href="/wiki/Dialog_systems" class="mw-redirect" title="Dialog systems">dialog system</a>.
</p>

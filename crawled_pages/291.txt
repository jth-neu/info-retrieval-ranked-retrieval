<p><b>GenieKnows</b> Inc., a <a href="/wiki/Privately_owned" class="mw-redirect" title="Privately owned">privately owned</a> <a href="/wiki/Vertical_search" title="Vertical search">vertical</a> <a href="/wiki/Search_engine" class="mw-redirect" title="Search engine">search engine</a> company based in <a href="/wiki/Halifax_Regional_Municipality" class="mw-redirect" title="Halifax Regional Municipality">Halifax</a>, <a href="/wiki/Nova_Scotia" title="Nova Scotia">Nova Scotia</a>. Like many <a href="/wiki/List_of_search_engines" title="List of search engines">internet search engines</a>, its revenue model centers on an <a href="/wiki/Online_advertising" title="Online advertising">online advertising</a> platform and <a href="/wiki/Business-to-business" title="Business-to-business">B2B</a> transactions. It focuses on a set of <a href="/wiki/Niche_market" title="Niche market">niche</a> search markets, or verticals, including health search, video games search, and <a href="/wiki/Local_search_(Internet)" title="Local search (Internet)">local business directory search</a>.<sup id="cite_ref-1" class="reference"><a href="#cite_note-1">&#91;1&#93;</a></sup>
</p>

<h2><span class="mw-headline" id="Technologies">Technologies</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=GenieKnows&amp;action=edit&amp;section=1" title="Edit section: Technologies">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>GenieKnows uses algorithms to download, assess, and categorize web pages and index them for later matching with <a href="/wiki/Web_search_query" title="Web search query">web search queries</a>.
</p>
<h3><span class="mw-headline" id="Local_Business_Directory_Search">Local Business Directory Search</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=GenieKnows&amp;action=edit&amp;section=2" title="Edit section: Local Business Directory Search">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>In 2005, GenieKnows entered the search engine market with a local business directory search engine.<sup id="cite_ref-2" class="reference"><a href="#cite_note-2">&#91;2&#93;</a></sup> Targeting only the United States in its <a href="/wiki/Beta_release" class="mw-redirect" title="Beta release">beta release</a>, the local search engine is similar to <a href="/wiki/Google_Maps" title="Google Maps">Google Maps</a> but uses the proprietary GeoRank algorithm to associate potentially uncategorized web pages containing addresses with businesses listed in an <a href="/wiki/Yellow_Pages" class="mw-redirect" title="Yellow Pages">internet Yellow Pages</a> directory by extracting addresses and <a href="/wiki/Geocoding" title="Geocoding">geocoding</a> these to identify <a href="/wiki/Geographic_coordinates" class="mw-redirect" title="Geographic coordinates">geographic coordinates</a> for which it associates the web page.<sup id="cite_ref-3" class="reference"><a href="#cite_note-3">&#91;3&#93;</a></sup>
</p><p>On February 29, 2008 <a rel="nofollow" class="external text" href="http://local.genieknows.com">GenieKnows Local</a> was launched as a completely revised local search engine extending beyond the 100 most populous US cities covered in its beta release. The local search engine utilizes processed municipal business data, road network data, national park data, and geocoding technology to provide localized search results ranked according to a business's relevance to a user's web query. As of the February 2008 release, the engine covers over 90% of Canadian and US municipalities with populations above 1000 residents.
</p><p>According to SEO and search marketing expert Jim Hedger, GenieKnows' strongest, most unusual and important product is its local search engine.<sup id="cite_ref-4" class="reference"><a href="#cite_note-4">&#91;4&#93;</a></sup>
</p>
<h3><span class="mw-headline" id="Vertical_Search_Engine">Vertical Search Engine</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=GenieKnows&amp;action=edit&amp;section=3" title="Edit section: Vertical Search Engine">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>GenieKnows entered the vertical search market in 2006 with a vertical search engine for <a href="/wiki/Video_games" class="mw-redirect" title="Video games">video games</a>-related web pages and another for health-related web pages.
</p><p>Web pages often describe or discuss a particular topic. In <a href="/wiki/Information_retrieval" title="Information retrieval">information retrieval</a> and <a href="/wiki/Machine_learning" title="Machine learning">machine learning</a> literature, classification algorithms have been used to automatically identify the subject matter of a web page. GenieKnows uses such algorithms as a <a href="/wiki/Focused_crawler" title="Focused crawler">focused crawler</a> to download <a href="/wiki/Web_pages" class="mw-redirect" title="Web pages">web pages</a>, identify pages that are on topic with the vertical, and <a href="/wiki/Index_(search_engine)" class="mw-redirect" title="Index (search engine)">index</a> and save those pages.
</p><p>The result is a search engine that contains only web pages that are on a given topic, tailoring to a niche market of web users who have an interest in a given topic, so all pages returned for a query will be on topic with the vertical being used. For revenue generation, the engine displays advertisements beside the search results from a network of advertisers where it receives <a href="/wiki/Pay_per_click" class="mw-redirect" title="Pay per click">pay per click</a>. GenieKnows is collaborating with <a href="/wiki/Yahoo!" title="Yahoo!">Yahoo!</a> to display targeted, contextualized advertisements to a targeted market of users.<sup id="cite_ref-5" class="reference"><a href="#cite_note-5">&#91;5&#93;</a></sup>
</p><p>In February 2008, GenieKnows added <a href="/wiki/Internet_community" class="mw-redirect" title="Internet community">online community</a> functionality to its vertical search engine. Users sharing interest in a topic can communicate and contribute content to the site in a manner similar, but on a smaller scale, to those of <a href="/wiki/Facebook" title="Facebook">Facebook</a>, and <a href="/wiki/Yahoo" class="mw-redirect" title="Yahoo">Yahoo</a>.
</p>

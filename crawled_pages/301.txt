<h1 id="firstHeading" class="firstHeading" lang="en">msnbot</h1>			<div id="bodyContent" class="mw-body-content">
				<div id="siteSub" class="noprint">From Wikipedia, the free encyclopedia</div>				<div id="contentSub"></div>
				<div id="jump-to-nav"></div>				<a class="mw-jump-link" href="#mw-head">Jump to navigation</a>
				<a class="mw-jump-link" href="#p-search">Jump to search</a>
				<div id="mw-content-text" lang="en" dir="ltr" class="mw-content-ltr"><div class="mw-parser-output"><p><span></span>
<b>msnbot</b> was a web-crawling robot (type of <a href="/wiki/Internet_bot" title="Internet bot">internet bot</a>), deployed by <a href="/wiki/Microsoft" title="Microsoft">Microsoft</a> to collect documents from the web to build a searchable index for the <a href="/wiki/MSN_Search" class="mw-redirect" title="MSN Search">MSN Search engine</a>. It went into beta in 2004, and had full public release in 2005. The month of October 2010 saw the official retirement of <b>msnbot</b> from most active web crawling duties and its replacement by <a href="/wiki/Bingbot" title="Bingbot">bingbot</a>.<sup id="cite_ref-1" class="reference"><a href="#cite_note-1">&#91;1&#93;</a></sup>
</p><p>As of September 2015 <b>msnbot</b> was still active from the <a href="/wiki/Microsoft" title="Microsoft">Microsoft</a> and the <a href="/wiki/Bing_(search_engine)" title="Bing (search engine)">Bing</a> webmaster help &amp; howto documentation still indicated that msnbot was active (but that it would retire soon).<sup id="cite_ref-2" class="reference"><a href="#cite_note-2">&#91;2&#93;</a></sup> The verification tool for bingbot<sup id="cite_ref-3" class="reference"><a href="#cite_note-3">&#91;3&#93;</a></sup> previously did not recognise <b>msnbot</b> IP addresses. A test executed on 2016-02-22 resulted in a yes: "Verdict for IP address 157.55.39.150: Yes - this IP address is a verified Bingbot IP address. Name: msnbot-157-55-39-150.search.msn.com."
</p>

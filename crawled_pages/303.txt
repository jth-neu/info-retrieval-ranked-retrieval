<h1 id="firstHeading" class="firstHeading" lang="en">Jughead (search engine)</h1>			<div id="bodyContent" class="mw-body-content">

<p><b>Jughead</b> is a <a href="/wiki/Search_engine" class="mw-redirect" title="Search engine">search engine</a> system for the <a href="/wiki/Gopher_(protocol)" title="Gopher (protocol)">Gopher protocol</a>. It is distinct from <a href="/wiki/Veronica_(search_engine)" title="Veronica (search engine)">Veronica</a> in that it searches a single <a href="/wiki/Server_(computing)" title="Server (computing)">server</a> at a time. 
</p><p>Jughead was developed by <a href="/w/index.php?title=Rhett_Jones&amp;action=edit&amp;redlink=1" class="new" title="Rhett Jones (page does not exist)">Rhett Jones</a> in 1993 at the <a href="/wiki/University_of_Utah" title="University of Utah">University of Utah</a>.
</p><p>The name "Jughead" was originally chosen to match the <a href="/wiki/Archie_search_engine" title="Archie search engine">Archie search engine</a>, as <a href="/wiki/Jughead_Jones" title="Jughead Jones">Jughead Jones</a> is Archie's friend in <a href="/wiki/Archie_Comics" title="Archie Comics">Archie Comics</a>. Later a <a href="/wiki/Backronym" title="Backronym">backronym</a> was developed: <i>Jonzy's Universal Gopher Hierarchy Excavation And Display</i>.
</p><p>It was released by the original author under the <a href="/wiki/GNU_General_Public_License" title="GNU General Public License">GNU General Public License</a> in 2006, and its <a href="/wiki/Source_code" title="Source code">source code</a> has been modernized to better run on current <a href="/wiki/POSIX" title="POSIX">POSIX</a> systems.
</p><p>Due to trademark issues, the modified version was called Jugtail, and has been made available for download on <a href="/wiki/GNU_Savannah" title="GNU Savannah">GNU Savannah</a> (see <a href="#External_links">external links</a>).
</p>

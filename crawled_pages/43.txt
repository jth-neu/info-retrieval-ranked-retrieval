<p><b>Cliqz</b> is a <a href="/wiki/Web_browser" title="Web browser">web browser</a> developed by Cliqz GmbH. It is a <a href="/wiki/Fork_(software_development)" title="Fork (software development)">fork</a> of the <a href="/wiki/Firefox" title="Firefox">Firefox</a> web browser with privacy-oriented changes, among which are a crowdsourced anti-tracking mechanism, and an in-house <a href="/wiki/Search_engine" class="mw-redirect" title="Search engine">search engine</a> embedded within the browser, utilizing its own index of web pages to produce suggestions within the address bar dropdown menu rather than on separate pages. The browser's developers argued that other privacy-focused search engines may still pass the user's <a href="/wiki/IP_address" title="IP address">IP address</a> to third-party search providers.<sup class="noprint Inline-Template Template-Fact" style="white-space:nowrap;">&#91;<i><a href="/wiki/Wikipedia:Citation_needed" title="Wikipedia:Citation needed"><span title="This claim needs references to reliable sources. (October 2017)">citation needed</span></a></i>&#93;</sup> Cliqz is available as a desktop and mobile web browser, as well as an extension for Firefox itself.<sup id="cite_ref-tc-ghosterybuy_3-0" class="reference"><a href="#cite_note-tc-ghosterybuy-3">&#91;3&#93;</a></sup><sup id="cite_ref-tc-mozcliqz_4-0" class="reference"><a href="#cite_note-tc-mozcliqz-4">&#91;4&#93;</a></sup>
</p><p>Cliqz is majority-owned by <a href="/wiki/Hubert_Burda_Media" title="Hubert Burda Media">Hubert Burda Media</a>. In August 2016, <a href="/wiki/Mozilla" title="Mozilla">Mozilla</a>, developer of Firefox, made a minority investment in Cliqz. Cliqz plans to eventually monetize the software through a program known as Cliqz Offers, which will deliver sponsored offers to users based on their interests and browsing history. However, these recommendations will be processed locally based on a remote repository of offers, with no personally identifiable data sent to remote servers.<sup id="cite_ref-tc-mozcliqz_4-1" class="reference"><a href="#cite_note-tc-mozcliqz-4">&#91;4&#93;</a></sup>
</p><p>On 15 February 2017, Cliqz International GmbH, a wholly owned subsidiary of Cliqz GmbH, acquired the privacy-oriented browser extension <a href="/wiki/Ghostery" title="Ghostery">Ghostery</a>.<sup id="cite_ref-5" class="reference"><a href="#cite_note-5">&#91;5&#93;</a></sup><sup id="cite_ref-6" class="reference"><a href="#cite_note-6">&#91;6&#93;</a></sup>
</p>
<h2><span class="mw-headline" id="Integration_with_Firefox">Integration with Firefox</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Cliqz&amp;action=edit&amp;section=1" title="Edit section: Integration with Firefox">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>On October 6, 2017, Mozilla announced a test where less than 1% of users downloading Firefox in Germany would receive a version with Cliqz software included. The feature provides recommendations directly in the browser's search field, for news, weather, sports, and other websites, based on the user's browsing history and activities. The press release noted that "Users who receive a version of Firefox with Cliqz will have their browsing activity sent to Cliqz servers, including the URLs of pages they visit," and that "Cliqz uses several techniques to attempt to remove sensitive information from this browsing data before it is sent from Firefox."<sup id="cite_ref-ZDNet_2017-10-09_7-0" class="reference"><a href="#cite_note-ZDNet_2017-10-09-7">&#91;7&#93;</a></sup> 
</p><p>According to the Firefox support website, this version of Firefox collects and sends data to the Cliqz corporation including text typed in the address bar, queries to other search engines, information about visited webpages and interactions with them including mouse movement, scrolling, and amount of time spent; and the user's interactions with the user interface of the Cliqz software. This data is tied to a unique identifier allowing Cliqz to track long-term performance. Interaction data collected and sent to the Mozilla corporation includes among other things, counts of visits to search engine pages, which search engines are used, and a Cliqz identifier. The data collection is enabled by default; users must actively opt-out if they do not wish the data to be transmitted.<sup id="cite_ref-Firefox_Support_8-0" class="reference"><a href="#cite_note-Firefox_Support-8">&#91;8&#93;</a></sup>
</p>

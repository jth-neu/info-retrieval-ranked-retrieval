<h1 id="firstHeading" class="firstHeading" lang="en">Gigablast</h1>			<div id="bodyContent" class="mw-body-content">

<p><b>Gigablast</b> is a web search engine founded in 2000. In 2015, it claimed to have indexed over 12 billion web pages, and received billions of queries per month.<sup id="cite_ref-2" class="reference"><a href="#cite_note-2">&#91;2&#93;</a></sup>
</p><p>The search engine <a href="/wiki/Source_code" title="Source code">source code</a> is written in the programming languages <a href="/wiki/C_(programming_language)" title="C (programming language)">C</a> and <a href="/wiki/C%2B%2B" title="C++">C++</a>. It was released as <a href="/wiki/Open-source_software" title="Open-source software">open-source software</a> under the <a href="/wiki/Apache_License" title="Apache License">Apache License</a> version 2, in July 2013.<sup id="cite_ref-3" class="reference"><a href="#cite_note-3">&#91;3&#93;</a></sup>
</p><p>Gigablast has provided, and provides, search results to other companies, such as <a href="/wiki/Ixquick" title="Ixquick">Ixquick</a>,<sup id="cite_ref-4" class="reference"><a href="#cite_note-4">&#91;4&#93;</a></sup> <a href="/wiki/Clusty" class="mw-redirect" title="Clusty">Clusty</a>,<sup id="cite_ref-5" class="reference"><a href="#cite_note-5">&#91;5&#93;</a></sup> <a href="/wiki/Zuula" title="Zuula">Zuula</a>, Snap,<sup id="cite_ref-6" class="reference"><a href="#cite_note-6">&#91;6&#93;</a></sup> and Blingo.
</p><p>In 2003, <i><a href="/wiki/The_New_York_Times" title="The New York Times">The New York Times</a></i> columnist Lee Dembart stated that "<i>Gigablast has its adherents</i>", but opined that <a href="/wiki/Google" title="Google">Google</a> is "<i>head and shoulders</i>" above it, and adds that Google's search results are more complete.<sup id="cite_ref-Dembart_7-0" class="reference"><a href="#cite_note-Dembart-7">&#91;7&#93;</a></sup>
</p>

<h2><span class="mw-headline" id="Features">Features</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Gigablast&amp;action=edit&amp;section=2" title="Edit section: Features">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Gigablast supports various specialized <a href="/wiki/Search_algorithm" title="Search algorithm">searches</a> and <a href="/wiki/Boolean_algebra" title="Boolean algebra">Boolean algebra</a> operators.<sup id="cite_ref-10" class="reference"><a href="#cite_note-10">&#91;10&#93;</a></sup> It also supports a related-concepts feature called <i>Giga Bits</i><sup id="cite_ref-11" class="reference"><a href="#cite_note-11">&#91;11&#93;</a></sup> and a blog-search feature.<sup id="cite_ref-12" class="reference"><a href="#cite_note-12">&#91;12&#93;</a></sup>
</p>
<h2><span class="mw-headline" id="History">History</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Gigablast&amp;action=edit&amp;section=3" title="Edit section: History">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><b>Gigablast</b> started as a small independent <a href="/wiki/Web_search_engine" title="Web search engine">web search engine</a> based in <a href="/wiki/New_Mexico" title="New Mexico">New Mexico</a>.<sup id="cite_ref-13" class="reference"><a href="#cite_note-13">&#91;13&#93;</a></sup> It was founded in 2000 by Matt Wells, formerly of <a href="/wiki/Infoseek" title="Infoseek">Infoseek</a>.
</p>

<p><b>HotBot</b> was a <a href="/wiki/Metasearch_engine" title="Metasearch engine">metasearch engine</a> for information on the <a href="/wiki/World_wide_web" class="mw-redirect" title="World wide web">world wide web</a>. It is now a privately-owned front end using <a href="/wiki/Bing_(search_engine)" title="Bing (search engine)">Bing search</a> for its search results.
</p>

<h2><span class="mw-headline" id="History">History</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=HotBot&amp;action=edit&amp;section=1" title="Edit section: History">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Hotbot was launched in May 1996 by <a href="/wiki/Wired_(magazine)" title="Wired (magazine)"><i>Wired</i> magazine</a>. It became a popular tool with search results served by the <a href="/wiki/Inktomi_(company)" class="mw-redirect" title="Inktomi (company)">Inktomi</a> database and directory results provided originally by <a href="/wiki/LookSmart" title="LookSmart">LookSmart</a> and then <a href="/wiki/DMOZ" title="DMOZ">DMOZ</a> since mid-1999. Hotbot also used search data from Direct Hit for a period, which was a tool that used click-through data to manipulate results. It was launched using a "new links" strategy of marketing, claiming to update its search database more often than its competitors. It also offered free webpage hosting, but only for a short time, and it was taken down without any notice to its users. It was one of the first search engines to offer the ability to search within search results.
</p><p><a href="/wiki/Lycos" title="Lycos">Lycos</a> acquired HotBot as part of its acquisition of Wired in 1998 and thereafter HotBot languished with limited development and falling market share. At the end of 2002 HotBot was relaunched as a multiple option search tool, giving users the option to search either the <a href="/wiki/Fast_Search_%26_Transfer" class="mw-redirect" title="Fast Search &amp; Transfer">FAST</a>, <a href="/wiki/Google" title="Google">Google</a>, <a href="/wiki/Inktomi_(company)" class="mw-redirect" title="Inktomi (company)">Inktomi</a> or <a href="/wiki/Teoma" title="Teoma">Teoma</a> databases.<sup id="cite_ref-2" class="reference"><a href="#cite_note-2">&#91;2&#93;</a></sup>
</p><p>Up to 2011, the HotBot website was merely a <a href="/wiki/Front_and_back_ends" title="Front and back ends">front end</a> for three third-party search engines (<a href="/wiki/Yahoo.com" class="mw-redirect" title="Yahoo.com">Yahoo.com</a>, <a href="/wiki/MSN_Search" class="mw-redirect" title="MSN Search">MSN</a>, and lyGo.com). In July 2011, HotBot was relaunched with a new robot-like mascot, a new logo, and a modern site design. In the beta, HotBot became a portal, returning not just web search results, but also searches from various Lycos websites, such as News, Shopping and <a href="/wiki/Weather_Zombie" title="Weather Zombie">Weather Zombie</a>. The portal interface lasted for roughly six months, and these features were instead reincorporated into the 2012 Lycos website redesign, returning HotBot to a simplified search interface.
</p><p>In October 2016, Lycos sold the Hotbot.com domain name for $155,000 to an unnamed buyer.<sup id="cite_ref-3" class="reference"><a href="#cite_note-3">&#91;3&#93;</a></sup>
</p>
<h2><span class="mw-headline" id="Toolbar_launch_in_2004">Toolbar launch in 2004</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=HotBot&amp;action=edit&amp;section=2" title="Edit section: Toolbar launch in 2004">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>In early 2004, Lycos launched a beta release of a free toolbar search product, Lycos HotBot DeskTop, which the company said was "the first product to integrate traditional desktop search with Web search within the browser." The HotBot DeskTop could search the Internet using Inktomi, e-mail folders for <a href="/wiki/Microsoft_Outlook" title="Microsoft Outlook">Microsoft Outlook</a> or <a href="/wiki/Outlook_Express" title="Outlook Express">Outlook Express</a>, and user documents stored on a hard drive. It also incorporated a blocker for pop-up ads and an RSS News Reader syndication. Indexes created to track e-mail and user files remained stored locally to protect user privacy. Text-based ads were displayed when viewing results for several types of Internet searches. Lycos licensed <a href="/wiki/DtSearch_Corp." class="mw-redirect" title="DtSearch Corp.">dtSearch</a> technology to power the local search options.<sup id="cite_ref-4" class="reference"><a href="#cite_note-4">&#91;4&#93;</a></sup>
</p>

<h1 id="firstHeading" class="firstHeading" lang="en">Mosaic (web browser)</h1>			<div id="bodyContent" class="mw-body-content">

<p><b>NCSA Mosaic</b>, or simply <b>Mosaic</b>, is the <a href="/wiki/Web_browser" title="Web browser">web browser</a> that popularized the World Wide Web and the Internet. It was also a client for earlier internet <a href="/wiki/Communications_protocol" class="mw-redirect" title="Communications protocol">protocols</a> such as <a href="/wiki/File_Transfer_Protocol" title="File Transfer Protocol">File Transfer Protocol</a>, <a href="/wiki/Network_News_Transfer_Protocol" title="Network News Transfer Protocol">Network News Transfer Protocol</a>, and <a href="/wiki/Gopher_(protocol)" title="Gopher (protocol)">Gopher</a>. The browser was named for its support of multiple internet protocols.<sup id="cite_ref-3" class="reference"><a href="#cite_note-3">&#91;3&#93;</a></sup> Its intuitive interface, reliability, <a href="/wiki/Microsoft_Windows" title="Microsoft Windows">Microsoft Windows</a> <a href="/wiki/Porting" title="Porting">port</a> and simple installation all contributed to its popularity within the web, as well as on Microsoft operating systems.<sup id="cite_ref-marc_4-0" class="reference"><a href="#cite_note-marc-4">&#91;4&#93;</a></sup> Mosaic was also the first browser to display images inline with text instead of displaying images in a separate window.<sup id="cite_ref-faq_5-0" class="reference"><a href="#cite_note-faq-5">&#91;5&#93;</a></sup> While often described as the first <a href="/wiki/Graphical_user_interface" title="Graphical user interface">graphical</a> web browser, Mosaic was preceded by <a href="/wiki/WorldWideWeb" title="WorldWideWeb">WorldWideWeb</a>, the lesser-known <a href="/wiki/Erwise" title="Erwise">Erwise</a><sup id="cite_ref-6" class="reference"><a href="#cite_note-6">&#91;6&#93;</a></sup> and <a href="/wiki/ViolaWWW" title="ViolaWWW">ViolaWWW</a>.
</p><p>Mosaic was developed at the <a href="/wiki/National_Center_for_Supercomputing_Applications" title="National Center for Supercomputing Applications">National Center for Supercomputing Applications</a> (NCSA)<sup id="cite_ref-faq_5-1" class="reference"><a href="#cite_note-faq-5">&#91;5&#93;</a></sup> at the <a href="/wiki/University_of_Illinois_at_Urbana%E2%80%93Champaign" title="University of Illinois at Urbana–Champaign">University of Illinois at Urbana–Champaign</a> beginning in late 1992. NCSA released the browser in 1993,<sup id="cite_ref-mosaic_and_th_w3_7-0" class="reference"><a href="#cite_note-mosaic_and_th_w3-7">&#91;7&#93;</a></sup> and officially discontinued development and support on January 7, 1997.<sup id="cite_ref-8" class="reference"><a href="#cite_note-8">&#91;8&#93;</a></sup> However, it can still be downloaded from NCSA.<sup id="cite_ref-9" class="reference"><a href="#cite_note-9">&#91;9&#93;</a></sup>
</p><p><a href="/wiki/Netscape_Navigator" title="Netscape Navigator">Netscape Navigator</a> was later developed by <a href="/wiki/Netscape" title="Netscape">Netscape</a> (originally known as Mosaic Communications Corporation), which employed many of the original Mosaic authors; however, it intentionally shared no code with Mosaic. Netscape Navigator's code descendant is <a href="/wiki/Firefox" title="Firefox">Mozilla Firefox</a>.<sup id="cite_ref-10" class="reference"><a href="#cite_note-10">&#91;10&#93;</a></sup> Starting in 1995 Mosaic lost market share to Netscape Navigator, and by 1997 only had a tiny fraction of users left, by which time the project was discontinued. <a href="/wiki/Microsoft" title="Microsoft">Microsoft</a> licensed Mosaic to create <a href="/wiki/Internet_Explorer" title="Internet Explorer">Internet Explorer</a> in 1995.
</p>

<h2><span class="mw-headline" id="Background">Background</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Mosaic_(web_browser)&amp;action=edit&amp;section=1" title="Edit section: Background">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="thumb tleft"><div class="thumbinner" style="width:202px;"><a href="/wiki/File:NCSAMosaic1.0Mac.png" class="image"><img alt="" src="//upload.wikimedia.org/wikipedia/en/thumb/a/a5/NCSAMosaic1.0Mac.png/200px-NCSAMosaic1.0Mac.png" width="200" height="150" class="thumbimage" srcset="//upload.wikimedia.org/wikipedia/en/thumb/a/a5/NCSAMosaic1.0Mac.png/300px-NCSAMosaic1.0Mac.png 1.5x, //upload.wikimedia.org/wikipedia/en/a/a5/NCSAMosaic1.0Mac.png 2x" data-file-width="365" data-file-height="273" /></a>  <div class="thumbcaption"><div class="magnify"><a href="/wiki/File:NCSAMosaic1.0Mac.png" class="internal" title="Enlarge"></a></div>Mosaic 1.0 running under <a href="/wiki/System_7" title="System 7">System 7.1</a>, displaying the Mosaic Communications Corporation (later <a href="/wiki/Netscape" title="Netscape">Netscape</a>) website.</div></div></div>
<p>After trying ViolaWWW, David Thompson demonstrated it to the NCSA software design group; <a href="/wiki/Marc_Andreessen" title="Marc Andreessen">Marc Andreessen</a>.<sup id="cite_ref-timbl_11-0" class="reference"><a href="#cite_note-timbl-11">&#91;11&#93;</a></sup> Andreessen and <a href="/wiki/Eric_Bina" title="Eric Bina">Eric Bina</a> originally designed and programmed NCSA Mosaic for Unix's <a href="/wiki/X_Window_System" title="X Window System">X Window System</a> called <i>xmosaic</i>.<sup id="cite_ref-faq_5-2" class="reference"><a href="#cite_note-faq-5">&#91;5&#93;</a></sup><sup id="cite_ref-mosaic_and_th_w3_7-1" class="reference"><a href="#cite_note-mosaic_and_th_w3-7">&#91;7&#93;</a></sup><sup id="cite_ref-timbl_11-1" class="reference"><a href="#cite_note-timbl-11">&#91;11&#93;</a></sup><sup id="cite_ref-Andreessen+Bina_1994_12-0" class="reference"><a href="#cite_note-Andreessen+Bina_1994-12">&#91;12&#93;</a></sup> Then, in December 1991, the <a href="/wiki/High_Performance_Computing_Act_of_1991" title="High Performance Computing Act of 1991">Gore Bill</a> created and introduced by then Senator and future Vice President <a href="/wiki/Al_Gore" title="Al Gore">Al Gore</a> was passed, which provided the funding for the Mosaic project. Development began in December 1992. Marc Andreessen announced the project on January 23, 1993.<sup id="cite_ref-13" class="reference"><a href="#cite_note-13">&#91;13&#93;</a></sup> The first alpha release (numbered 0.1a) was published in June 1993, and the first beta release (numbered 0.6b) followed quickly thereafter in September 1993. Ports to Microsoft Windows and <a href="/wiki/Macintosh" title="Macintosh">Macintosh</a> had been released in September.<sup id="cite_ref-timbl_11-2" class="reference"><a href="#cite_note-timbl-11">&#91;11&#93;</a></sup> A <a href="/wiki/AMosaic" title="AMosaic">port of Mosaic</a> to the <a href="/wiki/Commodore_Amiga" class="mw-redirect" title="Commodore Amiga">Commodore Amiga</a> was available by October 1993. NCSA Mosaic for Unix (X-Windows) version 2.0 was released on November 10, 1993.<sup id="cite_ref-14" class="reference"><a href="#cite_note-14">&#91;14&#93;</a></sup> Version 1.0 for Microsoft Windows was released on November 11, 1993.<sup id="cite_ref-The_History_of_NCSA_Mosaic_15-0" class="reference"><a href="#cite_note-The_History_of_NCSA_Mosaic-15">&#91;15&#93;</a></sup><sup id="cite_ref-16" class="reference"><a href="#cite_note-16">&#91;16&#93;</a></sup>  From 1994 to 1997, the <a href="/wiki/National_Science_Foundation" title="National Science Foundation">National Science Foundation</a> supported the further development of Mosaic.<sup id="cite_ref-17" class="reference"><a href="#cite_note-17">&#91;17&#93;</a></sup>
</p><p>Marc Andreessen, the leader of the team that developed Mosaic, left NCSA and, with <a href="/wiki/James_H._Clark" title="James H. Clark">James H. Clark</a>, one of the founders of <a href="/wiki/Silicon_Graphics,_Inc." class="mw-redirect" title="Silicon Graphics, Inc.">Silicon Graphics, Inc.</a> (SGI), and four other former students and staff of the <a href="/wiki/University_of_Illinois_at_Urbana%E2%80%93Champaign" title="University of Illinois at Urbana–Champaign">University of Illinois</a>, started Mosaic Communications Corporation. Mosaic Communications eventually became <a href="/wiki/Netscape_Communications_Corporation" class="mw-redirect" title="Netscape Communications Corporation">Netscape Communications Corporation</a>, producing <a href="/wiki/Netscape_Navigator" title="Netscape Navigator">Netscape Navigator</a>.
</p><p>1994 saw the first commercial product to incorporate Mosaic: <a href="/wiki/Santa_Cruz_Operation" title="Santa Cruz Operation">SCO</a> Global Access, a modified version of its <a href="/wiki/SCO_OpenServer" class="mw-redirect" title="SCO OpenServer">Open Desktop</a> version of Unix that served as an Internet gateway.<sup id="cite_ref-18" class="reference"><a href="#cite_note-18">&#91;18&#93;</a></sup>
</p><p><a href="/wiki/Spyglass,_Inc." title="Spyglass, Inc.">Spyglass, Inc.</a> licensed the technology and trademarks from NCSA for producing their own web browser but never used any of the NCSA Mosaic source code.<sup id="cite_ref-19" class="reference"><a href="#cite_note-19">&#91;19&#93;</a></sup> <a href="/wiki/Microsoft" title="Microsoft">Microsoft</a> licensed Spyglass Mosaic in 1995 for <a href="/wiki/United_States_dollar" title="United States dollar">US$</a>2 million, modified it, and renamed it <a href="/wiki/Internet_Explorer" title="Internet Explorer">Internet Explorer</a>.<sup id="cite_ref-win_20-0" class="reference"><a href="#cite_note-win-20">&#91;20&#93;</a></sup> After a later auditing dispute, Microsoft paid Spyglass $8 million.<sup id="cite_ref-win_20-1" class="reference"><a href="#cite_note-win-20">&#91;20&#93;</a></sup><sup id="cite_ref-21" class="reference"><a href="#cite_note-21">&#91;21&#93;</a></sup> The 1995 user guide <i>The HTML Sourcebook: The Complete Guide to HTML</i>, specifically states, in a section called <i>Coming Attractions</i>, that Internet Explorer "will be based on the Mosaic program".<sup id="cite_ref-The_HTML_Sourcebook_22-0" class="reference"><a href="#cite_note-The_HTML_Sourcebook-22">&#91;22&#93;</a></sup><sup class="reference" style="white-space:nowrap;">:<span>331</span></sup> Versions of Internet Explorer before <a href="/wiki/Internet_Explorer_7" title="Internet Explorer 7">version 7</a> stated "Based on NCSA Mosaic" in the About box. Internet Explorer 7 was audited by Microsoft<sup class="noprint Inline-Template Template-Fact" style="white-space:nowrap;">&#91;<i><a href="/wiki/Wikipedia:Citation_needed" title="Wikipedia:Citation needed"><span title="This claim needs references to reliable sources. (December 2007)">citation needed</span></a></i>&#93;</sup> to ensure that it contained no Mosaic code, and thus no longer credits Spyglass or Mosaic.
</p>
<h2><span class="mw-headline" id="Licensing">Licensing</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Mosaic_(web_browser)&amp;action=edit&amp;section=2" title="Edit section: Licensing">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>The licensing terms for NCSA Mosaic were generous for a proprietary software program. In general, non-commercial use was free of charge for all versions (with certain limitations). Additionally, the X Window System/Unix version publicly provided <a href="/wiki/Source_code" title="Source code">source code</a> (source code for the other versions was available after agreements were signed). Despite persistent rumors to the contrary, however, Mosaic was never released as <a href="/wiki/Open_source_software" class="mw-redirect" title="Open source software">open source software</a> during its brief reign as a major browser; there were always constraints on permissible uses without payment.
</p><p>As of  1993<sup class="plainlinks noexcerpt noprint asof-tag update" style="display:none;"><a class="external text" href="//en.wikipedia.org/w/index.php?title=Mosaic_(web_browser)&amp;action=edit">&#91;update&#93;</a></sup>, license holders included these:<sup id="cite_ref-The_Second_Phase_23-0" class="reference"><a href="#cite_note-The_Second_Phase-23">&#91;23&#93;</a></sup>

<h2><span class="mw-headline" id="Immediate_effect">Immediate effect</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Mosaic_(web_browser)&amp;action=edit&amp;section=3" title="Edit section: Immediate effect">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Other browsers existed during this period, notably <a href="/wiki/Erwise" title="Erwise">Erwise</a>, <a href="/wiki/ViolaWWW" title="ViolaWWW">ViolaWWW</a>, <a href="/wiki/MidasWWW" title="MidasWWW">MidasWWW</a> and <a href="/wiki/TkWWW" title="TkWWW">tkWWW</a>. These browsers, however, did not have the same effect as Mosaic on public use of the Internet.<sup id="cite_ref-24" class="reference"><a href="#cite_note-24">&#91;24&#93;</a></sup>
</p><p>In the October 1994 issue of Wired Magazine, Gary Wolfe notes in the article titled "The (Second Phase of the) Revolution Has Begun: Don't look now, but <a href="/wiki/Prodigy_(ISP)" class="mw-redirect" title="Prodigy (ISP)">Prodigy</a>, <a href="/wiki/AOL" title="AOL">AOL</a>, and <a href="/wiki/CompuServe" title="CompuServe">CompuServe</a> are all suddenly obsolete – and Mosaic is well on its way to becoming the world's standard interface":
</p>
<style data-mw-deduplicate="TemplateStyles:r856303468">.mw-parser-output .templatequote{overflow:hidden;margin:1em 0;padding:0 40px}.mw-parser-output .templatequote .templatequotecite{line-height:1.5em;text-align:left;padding-left:1.6em;margin-top:0}</style><blockquote class="templatequote"><p>When it comes to smashing a paradigm, pleasure is not the most important thing. It is the only thing. If this sounds wrong, consider Mosaic. Mosaic is the celebrated graphical "browser" that allows users to travel through the world of electronic information using a point-and-click interface. Mosaic's charming appearance encourages users to load their own documents onto the Net, including color photos, sound bites, video clips, and hypertext "links" to other documents. By following the links - click, and the linked document appears - you can travel through the online world along paths of whim and intuition. Mosaic is not the most direct way to find online information. Nor is it the most powerful. It is merely the most pleasurable way, and in the 18 months since it was released, Mosaic has incited a rush of excitement and commercial energy unprecedented in the history of the Net.<sup id="cite_ref-The_Second_Phase_23-1" class="reference"><a href="#cite_note-The_Second_Phase-23">&#91;23&#93;</a></sup>
</p></blockquote>
<h2><span class="mw-headline" id="Impact_of_Mosaic">Impact of Mosaic</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Mosaic_(web_browser)&amp;action=edit&amp;section=4" title="Edit section: Impact of Mosaic">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Mosaic was the web browser that led to the <a href="/wiki/Internet_boom" class="mw-redirect" title="Internet boom">Internet boom</a> of the 1990s. Robert Reid underscores this importance stating, "while still an undergraduate, Marc wrote the Mosaic software&#160;... that made the web popularly relevant and touched off the revolution".<sup id="cite_ref-Architects_of_the_Web_25-0" class="reference"><a href="#cite_note-Architects_of_the_Web-25">&#91;25&#93;</a></sup><sup class="reference" style="white-space:nowrap;">:<span>xlii</span></sup> Reid notes that Andreessen's team hoped:
</p>
<link rel="mw-deduplicated-inline-style" href="mw-data:TemplateStyles:r856303468"/><blockquote class="templatequote"><p>... to rectify many of the shortcomings of the very primitive prototypes then floating around the Internet. Most significantly, their work transformed the appeal of the Web from niche uses in the technical area to mass-market appeal. In particular, these University of Illinois students made two key changes to the Web browser, which hyper-boosted its appeal: they added graphics to what was otherwise boring text-based software, and, most importantly, they ported the software from so-called <a href="/wiki/Unix" title="Unix">Unix</a> computers that are popular only in technical and academic circles, to the [Microsoft] Windows operating system, which is used on more than 80 percent of the computers in the world, especially personal and commercial computers.<sup id="cite_ref-Architects_of_the_Web_25-1" class="reference"><a href="#cite_note-Architects_of_the_Web-25">&#91;25&#93;</a></sup><sup class="reference" style="white-space:nowrap;">:<span>xxv</span></sup>
</p></blockquote>
<p>Mosaic is not the first web browser for Microsoft Windows; this is <a href="/wiki/Thomas_R._Bruce" title="Thomas R. Bruce">Thomas R. Bruce</a>'s little-known <a href="/wiki/Cello_(web_browser)" title="Cello (web browser)">Cello</a>. The Unix version of Mosaic was already famous before the Microsoft Windows, Amiga, and Mac versions were released. Other than displaying images embedded in the text rather than in a separate window, Mosaic's original feature set is not greater than of the browsers on which it was modeled, such as ViolaWWW.<sup id="cite_ref-faq_5-3" class="reference"><a href="#cite_note-faq-5">&#91;5&#93;</a></sup> But Mosaic was the first browser written and supported by a team of full-time programmers, was reliable and easy enough for novices to install, and the inline graphics reportedly proved immensely appealing. Mosaic is said to have made the Web accessible to the ordinary person for the first time and already had 53% <a href="/wiki/Usage_share_of_web_browsers" title="Usage share of web browsers">market share</a> in 1995.<sup id="cite_ref-CiteSeerX_26-0" class="reference"><a href="#cite_note-CiteSeerX-26">&#91;26&#93;</a></sup>
</p><p>Reid also refers to <a href="/w/index.php?title=Matthew_K._Gray&amp;action=edit&amp;redlink=1" class="new" title="Matthew K. Gray (page does not exist)">Matthew K. Gray</a>'s website, <a rel="nofollow" class="external text" href="http://www.mit.edu/~mkgray/net/">Internet Statistics: Growth and Usage of the Web and the Internet</a>, which indicates a dramatic leap in web use around the time of Mosaic's introduction.<sup id="cite_ref-Architects_of_the_Web_25-2" class="reference"><a href="#cite_note-Architects_of_the_Web-25">&#91;25&#93;</a></sup><sup class="reference" style="white-space:nowrap;">:<span>xxv</span></sup>
</p><p>In addition, David Hudson concurs with Reid, noting that:
</p>
<link rel="mw-deduplicated-inline-style" href="mw-data:TemplateStyles:r856303468"/><blockquote class="templatequote"><p>Marc Andreessen's realization of Mosaic, based on the work of <a href="/wiki/Tim_Berners-Lee" title="Tim Berners-Lee">Berners-Lee</a> and the hypertext theorists before him, is generally recognized as the beginning of the web as it is now known. Mosaic, the first web browser to win over the Net masses, was released in 1993 and made freely accessible to the public. The adjective phenomenal, so often overused in this industry, is genuinely applicable to the... 'explosion' in the growth of the web after Mosaic appeared on the scene. Starting with next to nothing, the rates of the web growth (quoted in the press) hovering around tens of thousands of percent over ridiculously short periods of time were no real surprise.<sup id="cite_ref-Rewired_27-0" class="reference"><a href="#cite_note-Rewired-27">&#91;27&#93;</a></sup><sup class="reference" style="white-space:nowrap;">:<span>42</span></sup>
</p></blockquote>
<p>Ultimately, web browsers such as Mosaic became the <i><a href="/wiki/Killer_application" title="Killer application">killer applications</a></i> of the 1990s. Web browsers were the first to bring a graphical interface to search tools the Internet's burgeoning wealth of distributed information services. A mid-1994 guide lists Mosaic alongside the traditional, text-oriented information search tools of the time, <a href="/wiki/Archie_search_engine" title="Archie search engine">Archie</a> and <a href="/wiki/Veronica_(search_engine)" title="Veronica (search engine)">Veronica</a>, <a href="/wiki/Gopher_(protocol)" title="Gopher (protocol)">Gopher</a>, and <a href="/wiki/Wide_area_information_server" title="Wide area information server">WAIS</a><sup id="cite_ref-28" class="reference"><a href="#cite_note-28">&#91;28&#93;</a></sup> but Mosaic quickly subsumed and displaced them all. Joseph Hardin, the director of the NCSA group within which Mosaic was developed, said downloads were up to 50,000 a month in mid-1994.<sup id="cite_ref-29" class="reference"><a href="#cite_note-29">&#91;29&#93;</a></sup>
</p><p>In November 1992, there were twenty-six websites in the world<sup id="cite_ref-30" class="reference"><a href="#cite_note-30">&#91;30&#93;</a></sup> and each one attracted attention. In its release year of 1993, Mosaic had a What's New page, and about one new link was being added per day. This was a time when access to the Internet was expanding rapidly outside its previous domain of academia and large industrial research institutions. Yet it was the availability of Mosaic and Mosaic-derived graphical browsers themselves that drove the explosive growth of the Web to over 10,000 sites by Aug 1995 and millions by 1998.<sup id="cite_ref-31" class="reference"><a href="#cite_note-31">&#91;31&#93;</a></sup> Metcalfe expressed the pivotal role of Mosaic this way:
</p>
<link rel="mw-deduplicated-inline-style" href="mw-data:TemplateStyles:r856303468"/><blockquote class="templatequote"><p>In the Web's first generation, Tim Berners-Lee launched the Uniform Resource Locator (URL), Hypertext Transfer Protocol (HTTP), and HTML standards with prototype Unix-based servers and browsers. A few people noticed that the Web might be better than Gopher.
</p><p>In the second generation, Marc Andreessen and <a href="/wiki/Eric_Bina" title="Eric Bina">Eric Bina</a> developed NCSA Mosaic at the University of Illinois. Several million then suddenly noticed that the Web might be better than sex.
</p><p>
In the third generation, Andreessen and Bina left NCSA to found Netscape...</p><div class="templatequotecite">—&#8201;<cite>—Bob Metcalfe<sup id="cite_ref-32" class="reference"><a href="#cite_note-32">&#91;32&#93;</a></sup><sup id="cite_ref-Web_33-0" class="reference"><a href="#cite_note-Web-33">&#91;33&#93;</a></sup></cite></div>
</blockquote>
<h2><span class="mw-headline" id="End_of_Mosaic">End of Mosaic</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Mosaic_(web_browser)&amp;action=edit&amp;section=5" title="Edit section: End of Mosaic">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Mosaic's popularity as a separate browser began to lessen upon the release of Andreessen's <a href="/wiki/Netscape_Navigator" title="Netscape Navigator">Netscape Navigator</a> in 1994. This was noted at the time in <i>The HTML Sourcebook: The Complete Guide to HTML</i>: "Netscape Communications has designed an all-new WWW browser Netscape, that has significant enhancements over the original Mosaic program."<sup id="cite_ref-The_HTML_Sourcebook_22-1" class="reference"><a href="#cite_note-The_HTML_Sourcebook-22">&#91;22&#93;</a></sup><sup class="reference" style="white-space:nowrap;">:<span>332</span></sup>
</p><p>By 1998 its user base had almost completely evaporated, being replaced by other web browsers. After NCSA stopped work on Mosaic, development of the NCSA Mosaic for the X Window System source code was continued by several independent groups. These independent development efforts include mMosaic (multicast Mosaic)<sup id="cite_ref-34" class="reference"><a href="#cite_note-34">&#91;34&#93;</a></sup> which ceased development in early 2004, and Mosaic-CK and VMS Mosaic.
</p><p><b>VMS Mosaic</b>, a version specifically targeting <a href="/wiki/OpenVMS" title="OpenVMS">OpenVMS</a> operating system, was one of the longest-lived efforts to maintain Mosaic. Using the VMS support already built-in in original version (Bjorn S. Nilsson ported Mosaic 1.2 to VMS in the summer of 1993),<sup id="cite_ref-35" class="reference"><a href="#cite_note-35">&#91;35&#93;</a></sup> developers incorporated a substantial part of the HTML engine from mMosaic, another defunct flavor of the browser.<sup id="cite_ref-changehistory_36-0" class="reference"><a href="#cite_note-changehistory-36">&#91;36&#93;</a></sup> the last (4.2) release<sup class="plainlinks noexcerpt noprint asof-tag update" style="display:none;"><a class="external text" href="//en.wikipedia.org/w/index.php?title=Mosaic_(web_browser)&amp;action=edit">&#91;update&#93;</a></sup>, VMS Mosaic supported <a href="/wiki/HTML" title="HTML">HTML</a> 4.0, <a href="/wiki/OpenSSL" title="OpenSSL">OpenSSL</a>, <a href="/wiki/HTTP_cookie" title="HTTP cookie">cookies</a>, and various image formats including <a href="/wiki/Graphics_Interchange_Format" class="mw-redirect" title="Graphics Interchange Format">GIF</a>, <a href="/wiki/JPEG" title="JPEG">JPEG</a>, <a href="/wiki/Portable_Network_Graphics" title="Portable Network Graphics">PNG</a>, <a href="/wiki/BMP_file_format" title="BMP file format">BMP</a>, <a href="/wiki/Truevision_TGA" title="Truevision TGA">TGA</a>, <a href="/wiki/Tagged_Image_File_Format" class="mw-redirect" title="Tagged Image File Format">TIFF</a> and <a href="/wiki/JPEG_2000" title="JPEG 2000">JPEG 2000</a> image formats.<sup id="cite_ref-37" class="reference"><a href="#cite_note-37">&#91;37&#93;</a></sup> The browser works on <a href="/wiki/VAX" title="VAX">VAX</a>, <a href="/wiki/DEC_Alpha" title="DEC Alpha">Alpha</a>, and <a href="/wiki/Itanium" title="Itanium">Itanium</a> platforms.<sup id="cite_ref-VMSMosaic_38-0" class="reference"><a href="#cite_note-VMSMosaic-38">&#91;38&#93;</a></sup>
</p><p>Another long-lived version of Mosaic – <b>Mosaic-CK</b>, developed by <a href="/w/index.php?title=Cameron_Kaiser&amp;action=edit&amp;redlink=1" class="new" title="Cameron Kaiser (page does not exist)">Cameron Kaiser</a> – saw its last release (version 2.7ck9) on July 11, 2010; a maintenance release with minor compatibility fixes (version 2.7ck10) was released on 9 January 2015, followed by another one (2.7ck11) in October 2015.<sup id="cite_ref-CK_39-0" class="reference"><a href="#cite_note-CK-39">&#91;39&#93;</a></sup> The stated goal of the project is "<a href="/wiki/Lynx_(web_browser)" title="Lynx (web browser)">Lynx</a> with graphics" and runs on Mac OS X, Power <a href="/wiki/MachTen" title="MachTen">MachTen</a>, Linux and other compatible Unix-like <a href="/wiki/Operating_system" title="Operating system">OSs</a>.<sup id="cite_ref-CK_39-1" class="reference"><a href="#cite_note-CK-39">&#91;39&#93;</a></sup>
</p>
<h2><span class="mw-headline" id="Features">Features</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Mosaic_(web_browser)&amp;action=edit&amp;section=6" title="Edit section: Features">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<table class="plainlinks metadata ambox mbox-small-left ambox-content" role="presentation"><tbody><tr><td class="mbox-image"><a href="/wiki/File:Wiki_letter_w_cropped.svg" class="image"><img alt="[icon]" src="//upload.wikimedia.org/wikipedia/commons/thumb/1/1c/Wiki_letter_w_cropped.svg/20px-Wiki_letter_w_cropped.svg.png" width="20" height="14" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/1/1c/Wiki_letter_w_cropped.svg/30px-Wiki_letter_w_cropped.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/1/1c/Wiki_letter_w_cropped.svg/40px-Wiki_letter_w_cropped.svg.png 2x" data-file-width="44" data-file-height="31" /></a></td><td class="mbox-text"><div class="mbox-text-span">This section <b>needs expansion</b>. <small>You can help by <a class="external text" href="//en.wikipedia.org/w/index.php?title=Mosaic_(web_browser)&amp;action=edit&amp;section=">adding to it</a>.</small>  <small><i>(November 2010)</i></small></div></td></tr></tbody></table>
<p>Mosaic is based on the <a href="/wiki/Libwww" title="Libwww">libwww</a> <a href="/wiki/Library_(computing)" title="Library (computing)">library</a><sup id="cite_ref-40" class="reference"><a href="#cite_note-40">&#91;40&#93;</a></sup><sup id="cite_ref-41" class="reference"><a href="#cite_note-41">&#91;41&#93;</a></sup><sup id="cite_ref-42" class="reference"><a href="#cite_note-42">&#91;42&#93;</a></sup> and thus supported a wide variety of <a href="/wiki/Communications_protocol" class="mw-redirect" title="Communications protocol">Internet protocols</a> included in the library: <a href="/wiki/Archie_search_engine" title="Archie search engine">Archie</a>, <a href="/wiki/File_Transfer_Protocol" title="File Transfer Protocol">FTP</a>, <a href="/wiki/Gopher_(protocol)" title="Gopher (protocol)">gopher</a>, <a href="/wiki/Hypertext_Transfer_Protocol" title="Hypertext Transfer Protocol">HTTP</a>, <a href="/wiki/Network_News_Transfer_Protocol" title="Network News Transfer Protocol">NNTP</a>, <a href="/wiki/Telnet" title="Telnet">telnet</a>, <a href="/wiki/Wide_area_information_server" title="Wide area information server">WAIS</a>.<sup id="cite_ref-mosaic_and_th_w3_7-2" class="reference"><a href="#cite_note-mosaic_and_th_w3-7">&#91;7&#93;</a></sup>
</p>
